# Contact Management System - C# Console Application

A professional Contact Management System built with C# Console Application using Three-Tier Architecture, ADO.NET, and SQL Server integration.

## System Architecture

### Three-Tier Architecture Implementation
- **Presentation Layer**: Console Application UI
- **Business Logic Layer**: Contact operations and rules
- **Data Access Layer**: ADO.NET database operations

##  Database Design

### Contacts Table Structure
- Contact identification fields
- Personal information fields
- Communication details
- Country information fields

##  Technical Stack

### Backend Technologies
- **C# .NET** - Core programming language
- **ADO.NET** - Database connectivity
- **SQL Server** - Database management system
- **Console Application** - User interface

### Architecture Patterns
- **Three-Tier Architecture** - Separation of concerns
- **CRUD Operations** - Create, Read, Update, Delete

##  Core Features

### Contact Management
- **Add New Contacts** with complete information
- **View All Contacts** with formatted display
- **Update Contact Details** - modify existing records
- **Delete Contacts** - remove from database
- **Search Functionality** - find specific contacts

### Data Management
- **Comprehensive contact storage**
- **Organized data display**
- **Efficient data retrieval**
